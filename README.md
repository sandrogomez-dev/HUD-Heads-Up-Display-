# 🚀 JARVIS HUD - Advanced Holographic Interface

Un sistema de interfaz HUD (Heads-Up Display) cinematográfico inspirado en **Iron Man/JARVIS**, **Star Wars**, y **Pacific Rim**. Diseñado para ser **visualmente impresionante** y perfecto para demostración en redes sociales.

## ✨ Características Destacadas

### 🎬 Secuencia de Arranque
- **Arc Reactor animado** con anillos giratorios
- Barra de progreso de inicialización
- Mensajes del sistema en tiempo real
- Efectos de brillo y pulso

### 🌌 Efectos Visuales Espectaculares
- **Campo de partículas interactivo** con conexiones dinámicas
- **Círculos holográficos giratorios** en múltiples capas
- **Hexágonos flotantes** con animaciones 3D
- **Ondas de pulso** expandiéndose desde el centro
- **Efecto parallax** que sigue el mouse
- **Glass morphism** con blur en los paneles
- **Grid animado** de fondo

### 🎯 Sistema de Targeting Central
- **Retícula principal** con anillos concéntricos
- Brackets dorados animados
- Sistema de bloqueo de objetivos
- Crosshair holográfico
- Estados dinámicos ("ACQUIRING" → "TARGET LOCKED")

### 📊 Paneles de Información

#### Panel Superior Izquierdo - SYSTEM STATUS
- **3 anillos de progreso circulares** animados:
  - ⚡ **POWER CORE** (98%) - Cyan
  - 🛡️ **SHIELDS** (100%) - Azul
  - 🔫 **WEAPONS** (89%) - Naranja
- Actualizaciones en tiempo real
- Efectos de brillo y pulso
- Cambios de color según estado

#### Panel Superior Derecho - THREAT ANALYSIS
- **Sistema de escaneo holográfico**
- Línea de escaneo vertical animada
- Brackets en las esquinas con pulso
- Porcentaje de progreso en tiempo real
- Crosshair de targeting

#### Panel Inferior Izquierdo - RADAR 3D
- Radar circular con barrido 360°
- **5 contactos detectados** con pulso
- Información de rango (2.4 KM)
- Blips animados independientes
- Efecto de escaneo continuo

#### Panel Inferior Centro - FLIGHT DATA
- **Altímetro**: Números grandes animados
- **Velocímetro**: En km/h
- **Rumbo**: Orientación en grados
- Todos los valores cambian en tiempo real
- Efectos de glow en los números

#### Panel Inferior Derecho - NOTIFICATIONS
- **Feed de mensajes** del sistema
- Timestamps automáticos
- 3 tipos de alertas:
  - 🟢 **INFO** (verde)
  - 🟡 **WARNING** (amarillo)
  - 🔴 **DANGER** (rojo)
- Scroll automático
- Animación de entrada

### 🎨 Elementos Decorativos
- **4 esquinas** con marcos animados
- **Líneas horizontales** brillantes que se desplazan
- **Efecto scanline** sutil en toda la pantalla
- **Indicadores de estado** pulsantes

- Estructura semántica
- Canvas para partículas
- SVG para anillos de progreso

- **Grid & Flexbox** para layout
- **Keyframe animations** (más de 30 animaciones)
- **Backdrop filters** para efecto glass
- **CSS Variables** para colores
- **Transform 3D** para efectos de profundidad
- **Box shadows** para brillos
- **Clip-path** para hexágonos
- **Gradientes** complejos

### JavaScript
- **Canvas API** para partículas dinámicas
- **Generación procedural** de hexágonos
- **Sistemas de actualización** en tiempo real
- **Eventos del mouse** para parallax
- **Manipulación del DOM** dinámica
- **Intervalos** y **timeouts** coordinados

### Tipografía
- **Google Fonts**: Orbitron (futurista)
- Fallback a Courier New

## 🔧 Personalización

### Cambiar Colores
Edita las variables CSS en `styles.css`:
```css
:root {
    --primary-glow: #00f3ff;  /* Tu color aquí */
    --secondary-glow: #0af;
    /* etc... */
}
```

### Ajustar Velocidades de Animación
Busca `animation-duration` en el CSS y modifica los valores.

### Cambiar Mensajes
Edita el array `events` en `script.js`:
```javascript
const events = [
    { msg: 'Tu mensaje aquí', type: 'info' }
];
```
